## Quy tắc biên soạn

- File `00` ghi lại lịch sử đánh giá, nhận xét và điều chỉnh — **không phải** tài liệu yêu cầu chính thức.
- File `01` là tài liệu yêu cầu chính thức, giọng văn khách quan, chuẩn phân tích nghiệp vụ.
- Các file sau (`02-*`, `03-*`, ...) sẽ đi sâu vào từng khía cạnh cụ thể.
